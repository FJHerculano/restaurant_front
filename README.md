# Restaurant Front

Projeto frontend para restaurantes usando React + Vite.

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

## Estrutura

- `/src/Login.jsx` - Página de login
- `/src/Home.jsx` - Dashboard principal
- `/src/config.js` - Configuração da API
- `/src/App.jsx` - Rotas e autenticação

## API

Conecta na mesma API do owner-front: `http://localhost:3000/api`
